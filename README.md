## 🔧 使用するテスト技法

| テスト技法 | 適用箇所 |
|-----------|---------|
| 同値分割 | プラン選択、通話料金、SMS料金、オプション |
| 境界値分析 | 通話時間、SMS文字数、データ量、契約期間、日割り計算 |
| デシジョンテーブル | 通話料金、割引適用、契約解除料 |
| 状態遷移テスト | 割引期間、オプション無料期間、データ容量段階変化 |
| ペアワイズ法 | プラン組み合わせ |
| ユースケーステスト | E2Eシナリオ、初回請求合算 |
| エラー推測 | 異常系テスト |
